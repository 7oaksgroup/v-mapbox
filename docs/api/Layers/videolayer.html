# Props

# urls

  • Type: Array
  • Required
  • Non-Synced
  • Description: URLs to video content in order of preferred format.
  • See: urls in Mapbox Source Style Spec

# coordinates

  • Type: Array
  • Required
  • Non-Synced
  • Description: Corners of video specified in longitude, latitude pairs.
  • See: coordinates in Mapbox Source Style Spec
